## Sample1

サーバーを立ち上げただけ。

```
yarn sample1
```

- [http://localhost:1224/](http://localhost:1224/)にアクセス
- 「Hello World」が表示される

## Sample2

HTML を返すようにしたもの。

```
yarn sample2
```

- [http://localhost:1225/](http://localhost:1225/)にアクセス
- 「title, text」が表示される

## Sample3

ルーティング処理を追加。

```
yarn sample3
```

- [http://localhost:1226/](http://localhost:1226/)にアクセス
- 「最初のページ」が表示される

## Sample4

Sample3 の改善版。読み込んだ HTML ファイルをリユースする形。
